<DB에 공간 추가>
1.새 T/S 생성(=Datafile 생성)
S> create tablespace appuser
   datafile '/u01/app/oracle/oradata/ORCL/datafile/appuser01.dbf' size 100M
   autoextend on next 10M maxsize 500M;  

2.기존 T/S에 Datafile 추가
S> alter tablespace appuser
   add datafile '/u01/app/oracle/oradata/ORCL/datafile/appuser02.dbf' size 50M;

3.기존 Datafile의 사이즈 늘리기
S> alter database datafile '/u01/app/oracle/oradata/ORCL/datafile/appsuer01.dbf'
   resize 200M;

4.Datafile에 자동확장 기능 설정
(T/S 생성 시)
S> create tablespace appuser
   datafile '/u01/app/oracle/oradata/ORCL/datafile/appuser01.dbf' size 100M
   autoextend on next 10M maxsize 500M;  

(T/S 생성 후)
S> alter database datafile '/u01/app/oracle/oradata/ORCL/datafile/appuser02.dbf'
   autoextend on next 20M;
