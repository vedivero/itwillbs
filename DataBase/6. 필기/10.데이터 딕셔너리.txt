<Data Dictionary 구성요소>
1.Base Tables : DB의 모든 정보를 Oracle이 자기 보기 쉬운 형태로
		기록해 놓은 테이블

2.Data Dictionary Views : Base Table을 사용자가 보기 쉬운 형태로
                          만들어 놓은 뷰
			  읽기 전용(select만 가능)
 1)user_xxx : 내가 소유한 object에 대한 정보 조회
	      ex) user_tables, user_views, user_users,
		  user_constraints, user_indexes
 2)all_xxx : 내가 소유하거나 접근 가능한 object에 대한 정보 조회
	     ex) all_tables, all_view, all_indexes
 3)dba_xxx : DB의 모든 정보 조회, DBA만 select 가능함, 실시간 X
	     ex) dba_users, dba_data_files, dba_tables
 4)v$xxx : DB의 모든 정보 조회, DBA만 select 가능함, 실시간 O
	   ex) v$session, v$datafile
------------------------------------------------
[사용해야하는 Data Dictionary Views]
* user_constraints
* user_cons_columns

[출력] 
column_name, constraint_name, constraint_type,
search_condition, r_constraint_name

[조건]
테이블명 입력 받아야 함(대소문자 상관없이 출력 가능)
