
   클래스 : 추상화된 객체를 코드로 표시한 형태
   
   객체 : 내 눈앞에 보이는 모든 대상 -> 골라낸다(프로그래밍)=> 추상화
      -> 객체를 코드로 표시
      
   클래스 - 객체   =>  인스턴스화(객체화) : 메모리에 올린다 => new연산자
  붕어빵틀  - 붕어빵     => 가열 동작
  
  * 클래스 == 객체 ?  다르다.
  
  * 클래스의 구성요소
   1. 생성자 -> 멤버 변수를 초기화
   2. 멤버 변수 -> 객체의 속성
   3. 멤버 메서드 -> 객체의 동작
   
   
     휴대폰 객체
